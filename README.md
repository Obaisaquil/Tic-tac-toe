# Tic Tac Toe Game

A simple **Tic Tac Toe** game built using **HTML**, **CSS**, and **JavaScript**. It allows two players to play alternately with automatic win, draw, and reset functionality.

## Features
- Two-player gameplay
- Win and draw detection
- Reset game option
- Responsive and interactive UI

## Special Feature
- Prediction : It will predict draw matches

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the repository.
2. Open `index.html` in your web browser.
3. Start playing the game.
